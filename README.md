- 项目介绍：一个简洁的在线聊天网站，实现了基本的好友对话、好友上线提醒等功能
- 项目环境：jdk10+tomcat9.0
- 项目技术：springboot+websocket+bootstrap
- 项目工具：Idea、Maven
